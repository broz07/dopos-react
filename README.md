## Instalace

Stáhněte si zdrojový kód a nainstalujte závislosti.

```bash
git clone https://github.com/broz07/dopos-react.git
# and
cd dopos-react
# and
yarn
```

## Spuštění

### Vývojový server

```bash
yarn dev
```

### Produkční server

```bash
yarn fullstart
# or
yarn build && yarn start
```

Otevřete [http://localhost:3000](http://localhost:3000) ve svém prohlížeči.
